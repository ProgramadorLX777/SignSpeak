Proyecto SignSpeak

Proyecto desarrollado en Python mediante el uso de librerías OCV y MediaPipelines para el reconocimiento de Lengua de Señas Chilenas (LSCH). 

Para poder ejecutar el proyecto se requiere instalar de todas las librerías y dependencias en el ordenador y sobretodo solo funciona con la versión de Python 3.12, de ahí en adelante las librerías modernas no tienen publicado de manera oficial las dependencias preparadas para ejecutar este proyecto.
